# Contributing

Thanks for helping make this the best learning path for GPU-accelerated recommender systems!

## What belongs here

This is a **curated, opinionated** list — not an exhaustive dump. A resource belongs here if it genuinely helped someone learn one of the stages (P0-P6).

Good additions:
- A tutorial, book, course, paper, talk, or tool that taught you something well.
- A hands-on project or exercise that builds a specific skill.
- A correction or update to an existing entry (dead links, better alternatives).

Please avoid:
- Low-effort link dumps or SEO spam.
- Paywalled content without a clear note that it is paid.
- Duplicates of what is already listed.

## How to contribute

1. Fork the repository and create a branch: `git checkout -b add-resource-xyz`.
2. Add your link under the correct stage in `README.md`.
3. Use this format: a link followed by a one-line description of **why** it is good.
   - Example: `- [An Even Easier Introduction to CUDA](url) — best first read for the CUDA execution model`
4. Keep entries high-signal and concise.
5. Commit, push, and open a Pull Request describing what you added and why.

## Style

- One resource per line.
- Describe the *value*, not just the title.
- Prefer free/official sources where quality is equal.
- English for all entries.

## Code of conduct

Be respectful and constructive. We are all here to learn.
